Links to interesting open source projects

----------------------------------------------------------------------------------------------------
Audio Software

Ardour:
https://github.com/Ardour/ardour
-A fully featured DAW.

LMMS:
https://lmms.io/
https://github.com/LMMS/lmms
-Another DAW similar to FL

Qtractor:
https://qtractor.org/qtractor-index.html#Intro
-A DAW based on Qt for Linux.

Surge:
https://surge-synthesizer.github.io/
https://github.com/surge-synthesizer/surge
-A hybrid synthesizer (mostly VA but with some more exotic oscillator types). Very popular with 
 active community.
-https://surge-synth-team.org/ is a portal website with links to other projects of the Surge Synth
 Team

Element:
https://github.com/kushview/element
-A modular AU/LV2/VST/VST3 audio plugin host. Works standalone or as plugin. 

Vital:
https://github.com/mtytel/vital
-A spectral warping wavetable synthesizer with very visual GUI.
-Source code seems to be not get updated anymore since a while.
-Check out also Vitalium: ...

Distrho:
https://github.com/DISTRHO/DISTRHO-Ports
-open source project with the goal of making cross-platform audio plugins and GNU/Linux + LV2 ports.
-Includes Vitalium: https://github.com/DISTRHO/DISTRHO-Ports/tree/master/ports-juce6.0/vitalium

BespokeSynth:
https://github.com/BespokeSynth/BespokeSynth
-A Reaktor-style modular synthesizer

ZynAddSubFX / Zyn-Fusion:
https://zynaddsubfx.sourceforge.io/
https://zynaddsubfx.sourceforge.io/zyn-fusion.html
https://github.com/zynaddsubfx
-Hybrid additive/subtractive/pad-synth synthesizer
-Zyn-Fusion has an interesting business model: It's open source but you can buy a built version.

Cardinal:
https://github.com/DISTRHO/Cardinal
-A VCV-Rack-style modular synthesizer

Geonkick:
https://geonkick.org/
https://github.com/Geonkick-Synthesizer/geonkick
-A drum synthesizer with focus on bassdrums

Giada:
https://www.giadamusic.com/
-Not sure...seems to be some sort of tracker like music software with focus on live performance?

Carla:
https://kx.studio/Applications:Carla
-A modular audio plugin host. Works standalone only, I think (verify!).

Wavetable:
https://github.com/FigBug/Wavetable
-A 2 oscillator wavetable synth with flexible modulation options.

Plugins:
https://github.com/FigBug/SocaLabs
-Audio Plugins by FigBug - includes the "Wavetable" synth above. Other plugins are an organ, a 
 piano, Voc (vocal synth), mverb (Dattorro figure-8 reverb), emulations of: Commodore SID, Nintendo 
 RP2A03, Gameboy PAPU, Sega SN76489

Audio Programming Environment:
https://www.jthorborg.com/ape.html
https://bitbucket.org/Mayae/ape/src/master/
-An audio plugin, that allows to directly program C++ DSP in real time.
-https://www.kvraudio.com/forum/viewtopic.php?t=402054
 https://www.kvraudio.com/forum/viewtopic.php?f=33&t=560622

Cmajor:
https://github.com/cmajor-lang
-A C/C++ like domain specific language (DSL) for audio DSP.

Sfizz:
https://github.com/sfztools/sfizz
https://github.com/sfztools/sfizz-ui
-SFZ player

DtBlkFx:
https://github.com/dozius/DtBlkFx
-FFT based spectral processing plugin ("Darrel Tam's Block Effects?" ...probably)

EqualizerAPO:
https://equalizerapo.com/
https://sourceforge.net/projects/equalizerapo/
-System wide equalizer for Windows. Allows hosting of VST plugins, too.

Helm:
https://github.com/mtytel/helm
-Subtractive synthesizer 

Odin:
https://github.com/TheWaveWarden/odin2
-Subtractive Synthesizer

Dexed:
https://asb2m10.github.io/dexed/
-FM-Synthesizer, Yamaha DX-7 emulation

Airwindows Plugins:
https://www.airwindows.com/
https://github.com/airwindows
https://github.com/airwindows/airwindows
-A collection of plugins - GUI-less but does interesting/exotic stuff DSP-wise (including Kalman 
 filtering) so it might be inspirational

MDA Plugins:
https://sourceforge.net/projects/mda-vst/
https://drobilla.net/software/mda-lv2.html
https://gitlab.com/drobilla/mda-lv2
https://github.com/hollance/mda-plugins-juce
-Another collection of GUI-less plugins

CMT (Computer Music toolkit) Plugins:
https://www.ladspa.org/cmt/plugins.html
https://www.ladspa.org/download/index.html
-Yet another collection of (GUI-less?) plugins

Freeverb:
https://github.com/sinshu/freeverb
https://www.dsprelated.com/freebooks/pasp/Freeverb.html
-Schroeder/Moorer allpass based reverb (public domain)

Aether:
https://github.com/Dougal-s/Aether
-Algorithmic reverb plugin for LV2

Cloudseed:
https://github.com/ValdemarOrn/CloudSeed
-The DSP algo on which Aether is based.

quantumVerb
https://qvbdev.github.io/quantumVerb/
https://github.com/QVbDev/quantumVerb
-Convolution reverb

KlangFalter
https://github.com/HiFi-LoFi/KlangFalter
-Convolution reverb

https://github.com/robbert-vdh/nih-plug/
-API-agnostic audio plugin framework written in Rust, as well as a small collection of plugins.

ZL-Audio Plugins
https://github.com/ZL-Audio
https://github.com/ZL-Audio/ZLEqualizer
-Some plugins made with JUCE. The EQ appears to have a nice visual and modern GUI and uses the
 Vicanek design formulas. It's dynamic as well.

WayVerb
https://reuk.github.io/wayverb/
https://github.com/reuk/wayverb
-Raytracing based impulse response generator

Libraries and SDKs

Brickworks:
https://www.orastron.com/brickworks
https://www.kvraudio.com/forum/viewtopic.php?t=589519
-Audio DSP library available for licensing under GPL or commercially

Synthesis Toolkit (STK):
https://ccrma.stanford.edu/software/stk/
https://github.com/thestk/stk
-Audio DSP library

Audio Toolkit:
https://github.com/AudioTK
https://github.com/AudioTK/AudioTK
-Audio DSP library

https://github.com/nothings/stb/
-Single-file public domain (or MIT licensed) libraries for C/C++.
-Has image (.png,etc.) read/write, .ttf-rendering

CLAP-SDK and related stuff:
https://github.com/free-audio
-Liberally licensed audio plugin format.
-Tutorial: https://www.kvraudio.com/forum/viewtopic.php?t=589258

Avendish:
https://github.com/celtera/avendish/
-Solution to the quadratic glue MxN problem: With M plugin formats and N plugins we need to write 
 and maintain(!) an amount of O(M*N) boilerplate glue code. Avendish claims to solve this 
 maintenance nightmare.

SA-Toolkit:
https://github.com/skei/SA_Toolkit
-Plugin framework by Tor Helge Skei

Dust-Toolkit:
https://github.com/signaldust/dust-toolkit
-Application and GUI toolkit by mystran, includes font rendering 

Anti Grain Geometry
https://github.com/ghaerr/agg-2.6
-Rasterization based 2D graphics rendering library.

Dear ImGUI
https://www.dearimgui.com/
https://github.com/ocornut/imgui
-Immediate mode GUI framework

Elements
https://github.com/cycfi/elements
-Lightweight GUI framework

Tracktion Engine
https://www.tracktion.com/develop/tracktion-engine
https://github.com/Tracktion/tracktion_engine/
-Audio engine of the Tracktion DAW

JUCE
https://juce.com/
https://github.com/juce-framework/JUCE
-Framework for building audio software and plugins

IPlug2
https://iplug2.github.io/
https://github.com/iPlug2/iPlug2
-Another audio plugin framework. More lightweight and more liberally licensed than JUCE.

WDL
https://www.cockos.com/wdl/
-GUI-library plus some audio related stuff

Choc
https://github.com/Tracktion/choc/
-Julian Storer's new, small, header-only library

Farbot (FAbian's Realtime Box o' Tricks)
https://github.com/hogliux/farbot
-Library for thread-sync tasks typical in audio software programming

Spleeter
https://github.com/deezer/spleeter
-AI based source separation

Demucs
https://github.com/facebookresearch/demucs
-AI based source separation
